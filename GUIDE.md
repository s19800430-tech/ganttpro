## 1. 快速开始

```
1. 用浏览器打开 gantt.html（推荐 Chrome / Edge）
2. 输入你的姓名 → 点击「开始使用」
3. 点击「+ 建立第一个项目」或从「文件 → 新建项目」选择模板
4. 开始添加任务！
```

> 💡 **离线使用**：打开方式 `file:///path/to/gantt.html`，无需联网（导出 PDF/PNG 需联网加载 html2canvas）。

## 3. 创建项目

### 3.1 新建空白项目

1. 菜单 **文件 → 新建项目**（或 `Ctrl+Shift+N`）
2. 填写项目名称、开始日期、颜色
3. 点击「建立项目」

### 3.2 使用模板

点击**套用模板**下拉框，选择内置模板：

| 模板名称 | 适用场景 |
|----------|----------|
| 硬件产品开发与量产标准流程 | 从需求定义到 MP 量产的完整硬件研发周期 |
| **非标设备项目管理流程** ⭐ | 立项 → 设计评审 → 采购 → 组装调试 → 现场实施 → 终验收 |

> 模板会自动计算所有任务日期，从你指定的项目开始日期顺延排列。

## 4. 任务管理

### 4.1 新增任务

**方式一**：工具栏点击「＋ 新增任务」  
**方式二**：菜单「插入 → 普通任务」  
**方式三**：任务行末尾的「＋」按钮（行内追加）  
**方式四**：快捷键 `Ctrl+Enter`

### 4.2 内联编辑（表格直接输入）

无需打开弹窗，直接点击单元格即可编辑：

| 单元格 | 操作 |
|--------|------|
| **任务名称** | 单击 → 直接输入，Enter 确认 |
| **开始日期** | 单击 → 日期选择器弹出 |
| **结束日期** | 单击 → 日期选择器弹出 |
| **工作天数** | 单击 → 输入数字，自动推算结束日 |
| **负责人** | 单击 → 输入姓名，多人用逗号分隔 |
| **前置任务** | 单击 → 输入行号，支持 FS/SS/FF/SF 类型 |

### 4.3 任务类型

| 类型 | 图示 | 说明 |
|------|------|------|
| 普通任务 | `████████` | 有工期的工作项 |
| 里程碑 | `◆` | 零工期的关键节点 |
| 群组 | `◄████████►` | 折叠/展开子任务的父级汇总 |

### 4.4 快速复制 / 移动

- **拖拽行** 左侧 `⋮⋮` 图标可重新排序
- `Ctrl+D` 复制当前选中任务
- 任务行右侧操作按钮：缩进 `→` / 取消缩进 `←` / 删除 `🗑`

## 5. 里程碑

里程碑在甘特图中显示为 **菱形 ◆**，代表项目关键节点（如评审、交付、发货等）。

### 5.1 两种创建方式

**方式一：插入菜单**
```
菜单「插入 → 里程碑」→ 填写名称和日期
```

**方式二：零工期自动转换** ⭐
```
将任务的「开始日期」和「结束日期」设为同一天
→ 甘特图自动显示为菱形里程碑
```

> 💡 这是最快的方式：先建普通任务，日期填同一天，立即变成菱形。

### 5.2 里程碑状态

| 状态 | 显示 |
|------|------|
| 未完成 | 彩色菱形 ◆ |
| 已完成 | 灰色透明 ◇ |
| 逾期 | 红色菱形 ◆（配合红色警告） |

## 6. 前置依赖

### 6.1 依赖类型

| 类型 | 说明 | 输入格式 |
|------|------|----------|
| **FS**（完成→开始）| A 完成后 B 才能开始 | `3` 或 `3FS` |
| **SS**（开始→开始）| A 开始后 B 才能开始 | `3SS` |
| **FF**（完成→完成）| A 完成后 B 才能完成 | `3FF` |
| **SF**（开始→完成）| A 开始后 B 才能完成 | `3SF` |
| **带延迟** | A 完成后 3 个工作日再开始 B | `3FS+3` |

### 6.2 设置方法

1. 点击任务行的「前置任务」单元格
2. 输入前置任务的**行号**（如 `2`、`2SS`、`3FS+2`）
3. 或在弹出的**依赖选择器**中勾选

> 设置依赖后，系统自动重新排程所有后续任务日期。

## 7. 负责人与资源

### 7.1 任务表格快速输入

点击任务行的「**负责人**」列，直接输入姓名：
- 单人：`张三`
- 多人：`张三, 李四, 王五`
- 系统会自动生成彩色头像缩写并显示在行内

### 7.2 弹窗详细设置

双击任务行（或「插入 → 编辑任务」）打开完整表单，「负责人」字段支持芯片（Chip）多选输入：
1. 在输入框输入姓名
2. 按 **Enter** 或 **逗号** 添加为芯片标签
3. 点击芯片上的 `×` 删除
4. **Backspace** 删除最后一个芯片

### 7.3 工作量视图

切换「视图 → 工作量」，左侧面板按负责人汇总工作量，右侧显示每人的任务分布热图。

## 8. 甘特图操作

### 8.1 拖拽操作

| 操作 | 效果 |
|------|------|
| **拖拽条形中部** | 移动任务日期（整体平移） |
| **拖拽条形右端** | 调整工期（结束日延长/缩短） |
| **拖拽里程碑菱形** | 移动里程碑日期 |
| **点击空白日期格** | 在该日期创建新任务 |

### 8.2 缩放时间轴

| 按钮 | 视图 | 适用场景 |
|------|------|----------|
| Day | 日视图 | 短期精细计划（≤2周） |
| Week | 周视图（默认） | 中期项目（1~3月） |
| Month | 月视图 | 长期项目（3月以上） |

滚轮 + Ctrl 也可缩放；鼠标拖拽甘特图区域可左右平移。

### 8.3 今日标记

红色竖线 = 今天。甘特图启动时自动滚动到今日位置。

## 10. 非标项目模板示例

以下是**非标设备项目管理**的标准周期，可通过「文件 → 新建项目 → 套用模板 → 非标设备项目管理流程」一键生成：

```
◆ KOM 启动会（立项里程碑）
│
├── 设计阶段
│   ├── 概念设计（任务）
│   ├── ◆ 概设评审（里程碑）
│   ├── 详细设计（任务）
│   └── ◆ 详设评审（里程碑）
│
├── 采购阶段
│   ├── 物料清单确认
│   ├── 供应商询价与下单
│   └── 物料到货验收
│
├── 组装调试
│   ├── 机械结构组装
│   ├── 电气接线
│   ├── 系统调试
│   └── ◆ 通电里程碑
│
├── 出厂验收
│   ├── 内部验收测试
│   ├── 客户出厂验收
│   └── ◆ 验收完成里程碑
│
├── 打包发货
│   ├── 包装与装箱
│   ├── 物流运输
│   └── ◆ 发货里程碑
│
└── 现场实施
    ├── 设备吊装就位
    ├── 现场接线与配置
    ├── 现场调试与试运行
    ├── ◆ 恢复通电里程碑
    ├── 初步验收
    └── ◆ 终验收里程碑
```

> 模板共 **7 个阶段、20+ 个任务、9 个里程碑**，所有任务已配置前置依赖，套用后自动排期。

### 里程碑说明

| 里程碑 | 含义 | 触发条件 |
|--------|------|----------|
| **KOM 启动会** | 项目正式启动 | 立项申请批准 + 资源评估完成 |
| **概设评审** | 概念设计通过评审 | 概念设计完成 |
| **详设评审** | 详细设计通过评审 | 详细设计完成 |
| **通电里程碑** | 设备通电成功 | 组装调试全部完成 |
| **验收完成** | 客户出厂验收通过 | 内外部验收测试均通过 |
| **发货里程碑** | 设备离厂 | 包装完成 |
| **恢复通电** | 现场设备上电运行 | 现场接线调试完成 |
| **终验收** | 项目正式交付 | 初步验收通过 |

## 11. 导出与分享

### 11.1 导出格式

| 格式 | 菜单位置 | 说明 |

## 常见问题

**Q: 数据存在哪里？**  
A: 浏览器 IndexedDB（本地），关闭标签页不丢失。换浏览器或清除浏览器数据会丢失，建议定期「文件 → 导出 JSON」备份。

**Q: 任务日期为什么不能手动修改？**  
A: 有前置依赖的任务日期由系统自动排程决定。如需固定日期，勾选「固定开始日期」（小图钉图标）。

**Q: 怎么让任务显示为菱形里程碑？**  
A: 将开始日期和结束日期设为**同一天**，甘特图自动显示菱形。

**Q: 移动端能用吗？**  
A: 支持！手机浏览器访问时自动切换为移动视图，顶部显示项目名称，底部有任务/添加/报告/设置四个标签。弹窗以底部抽屉样式呈现。
